Utilisation

Bonjour, voici un guide pour vous permettre de comprendre notre outil !
Comme vous pouvez le voir, notre interface est découpée en 3 barres de navigation :

  • la barre d'outils (à gauche)
  • la barre d'édition (en haut)
  • la barre de save (à droite)

Celles si vous seront détaillées ci-dessous.

Barre d'outils

Notre barre d'outils se sépare en plusieurs fonctionnalités :

  • Efface tout les élèments (textes / formes) présents sur l'écran. Afin de supprimer un element uniquement , double cliquez dessus .
  • Crée un carré. Il faut d'abord renseigner les dimensions du carré dans la barre d'édition.
  • Crée un cercle. Vous pouvez définir sa taille en lui donnant le radius souhaité.
  • Crée un triangle. Il faut d'abord renseigner les dimensions du triangle dans la barre d'édition.
  • Crée une zone de texte dans laquelle vous pouvez écrire.

Barre d'édition

Ici vous retrouverez toutes les options permettant de personnaliser vos formes ainsi que vos textes (elles vous seront expliquées de gauche à droite) :

  • Police : Choisissez parmi les 3 polices (Serif, Courrier, Comic Sans MS ) celle que vous souhaitez.

  • Style : Choisissez le style de votre texte entre "italic", "oblique" et "normal".

  • Taille : Vous permet de choisir la taille de votre texte avant de créer une zone de texte.

  • Texte / Surligner : Permet d'appliquer une couleur au texte ou au surlignage. Pour cela cochez la/les case(s) souhaitée(s) puis sélectionnez la couleur voulue avant de cliquer sur la zone de texte que vous souhaitez modifier

  • Intérieur / Contour : Permet d'appliquer une couleur à l'interieur au ou contour d'une forme. Pour cela cochez la/les case(s) souhaitée(s) puis sélectionnez la couleur voulue avant de cliquer sur la forme que vous souhaitez modifier

  • Couleur : La palette permet de changer la couleur des textes et des formes. Pour cela se réferer à "Texte / Surligner" pour la couleur du texte et "Intérieur / Contour" pour la couleur des formes.

  • Width / Height : Renseignez les champs avec respectivement la hauteur et la largeur de la forme. Il faut préciser les dimensions avant de créer un Carré ou un Triangle.

  • Radius : Indiquez le radius pour modifier le rayon le cercle correspondant

Barre de save

Cette barre vous permet d'enregistrer votre chef d'oeuvre en devenir mais aussi d'en importer un autre. Il vous permet aussi de l'enregistrer soit sous format PDF soit sous format PNG.

  • Permet de sauvegarder votre travail dans un fichier.
  • Permet de charger des formes / textes precédemment sauvegardés.
  • Permet d'enregistrer la page sous format pdf. Pour utiliser l'exportation en pdf/png il vous faudra "node modules" et "vendor"
  • Créer une capture d'écran de la page et l'enregistre sous format png. Pour utiliser l'exportation en pdf/png il vous faudra "node modules" et "vendor"
